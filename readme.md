# dotfiles

### cp

> copy these files manually to prevent any unexpected consequences

```bash
$ cp .gitconfig ~/.gitconfig
$ cp .aliases ~/.aliases
$ cp .exports ~/.exports
$ cp .path ~/.path
$ cp .zshrc ~/.zshrc
$ cp .vimrc ~/.vimrc
```

### reload

```bash
$ source ~/.zshrc
```

### initial run

#### mac
